# Requirements
* Python 3 (>= 3.3)
* pySFML (for Python 3)

# Usage
Run in single-computer mode:

```shell
python3 "src/main.py"
```

Create a server:

```shell
python3 "src/main.py" create localhost 23324
```

Join the game:

```shell
python3 "src/main.py" join localhost 23324
```

Enjoy it!
